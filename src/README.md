Project Overview: Music management web app that allows users to manage their playlist and artists to manage their songs
Code was written following javaguides tutorial series: https://www.javaguides.net/2019/03/jsp-servlet-jdbc-mysql-crud-example-tutorial.html https://www.javaguides.net/2019/03/login-form-using-jsp-servlet-jdbc-mysql-example.html
Instructions: Download Apache Tomcat 9.0 on Eclipse IDE for Enterprise Java and Web Developers, Install dependencies standard-1.1.2, mysql-connector-java-8.0.12, servlet-api-2.5, jstl-1.2, jsp-api-2.2, add to build path in lib as well as lib in tomcat folder on hard drive. Start server as a service on windows 10 with MySQL workbench 8.0 CE. Run on server in Eclipse, select tomcat as server. Open browser and go to http://localhost:8080/dbproj/login.jsp.
